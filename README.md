## 🚀 Features

### User Management
- 🧑‍💼 User Registration & Login (JWT Authentication)
- 🔍 Search users by name or email
- ➕ Follow/Unfollow other users
- 🧑‍🤝‍🧑 View following/followers list

### Real-time Chat
- 💬 One-to-one messaging
- 📂 Group chats
- ✍️ Typing indicators
- 🔔 Online status
